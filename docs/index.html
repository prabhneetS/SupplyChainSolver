Model Logic Used

  • The dashboard follows the notebook flow: freight cost divided by 1,000 plus manufacturing variable cost creates the total variable cost matrix.
  • The optimization minimizes fixed plant-opening cost plus production flow cost while meeting demand in each market.
  • Capacity constraints restrict production based on whether a low-capacity or high-capacity plant is opened at each location.
  • No warehouse allocation, supplier scorecard, or fictional route data has been added because those fields were not present in the uploaded datasets.
